Contributing to CS 1.6 OpenGL Hook Project

Thank you for your interest in contributing!

How to Build the Project

Open Dev-C++ (or any C++ IDE that supports DLL projects).

Create a New Project.

Select Dynamic Link Library (DLL).

Set the compiler target to:

Release

32-bit (x86)

Add all source files from the /src folder into the project.

Make sure the output type is set to .dll.

Build the project.

The compiled DLL will be generated inside the project's output folder.

Contribution Rules

Keep the code clean and readable.

Avoid unnecessary libraries.

Follow consistent formatting.

Clearly explain what your pull request changes.

Do not submit unrelated changes.

How to Submit Changes

Fork the repository.

Create a new branch.

Make your changes.

Commit with a clear message.

Open a Pull Request describing your changes.
