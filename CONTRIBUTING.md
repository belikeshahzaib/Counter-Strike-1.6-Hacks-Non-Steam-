# Contributing to CS 1.6 OpenGL Hook Project

Thank you for your interest in contributing!

## How to Build the Project

1. Open **Dev-C++** (or any C++ IDE that supports DLL projects).
2. Create a **New Project**.
3. Select **Dynamic Link Library (DLL)**.
4. Set the compiler target to:

   * **Release**
   * **32-bit (x86)**
5. Add all source files from the `/src` folder into the project.
6. Make sure the output type is set to **.dll**.
7. Build the project.

The compiled DLL will be generated inside the project's output folder.

---

## Contribution Rules

* Keep the code clean and readable.
* Avoid unnecessary libraries.
* Follow consistent formatting.
* Clearly explain what your pull request changes.
* Do not submit unrelated changes.

---

## How to Submit Changes

1. Fork the repository.
2. Create a new branch.
3. Make your changes.
4. Commit with a clear message.
5. Open a Pull Request describing your changes.

---

If you are adding a new feature, please open an Issue first to discuss it.
